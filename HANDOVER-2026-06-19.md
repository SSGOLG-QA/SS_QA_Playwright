## 5. 미해결 F/U (계정/협의 선결)

| F/U | 선결조건 | 후속 작업 |
|---|---|---|
| 병렬 라이브 실증 | 테스트 계정 N개 (QA/인프라) | `$env:ACCOUNT_COUNT="N"; npm run auth` → `test:admin` + 코드 커밋 |
| `test:lang` 풀 병렬 | (위와 동일) | 언어/메뉴별 `test()` 분할 (현재 1 test=1 worker) |
| MCP PoC 실증 | 전용 계정 `admin-1.json` | Claude 재시작 → 드리프트 라이브 탐색 |
| Agentic 옵션 매트릭스 | 옵션→메뉴 게이팅 실제값 (QA/PM) | `IA_TREE` 옵션 태그 + 분류기 PoC (킹즈락 단독도 `미구현` 모호성 해소 가치) |

| 크로스플랫폼 E2E | 개발팀 API 계약 + teardown | API 경계 분할 설계 |
