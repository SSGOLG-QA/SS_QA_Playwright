> ℹ️ 별도 [저장] 버튼이 카드 내부에 미검출(`saveBtns:[]`) → 선택 즉시 반영(auto-save) 추정. **변경 동작 자체를 안 하므로 저장 경로 무관**(필요 시 추후 probe).

## 비파괴 / ⚠️ 주의
- **모든 액션이 파괴적**: `F&B 데이터 동기화`·`POS 메뉴 동기화`는 실제 동기화 실행, vue-select는 기본식당 변경 → **클릭/선택 금지, 노출·활성만 검증**.
- 현재버전 숫자(`811▲1`)는 데이터 의존 → 패턴 검증.
- 코스 수·식당명은 골프장 의존 → `≥1`/존재 검증.
- SNB/2뎁스 토글(`snb-toggle__switchbtn`, `depth-2-toggle__switchbtn1~13`)은 **네비게이션 공통 chrome**(Home~계정 관리 13개 대메뉴 토글) — 본 화면 검증 대상 **아님**.
- `gotoMenu` 진입 시 자동 RAW 스캔(오타/미가공코드) 적용.
