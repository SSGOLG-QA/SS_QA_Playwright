## Locator 전략 (리뉴얼)

> ⚠️ td17 실제 href/class 미분석 상태. 진입 후 Console 추출로 확정 필요.
> 그 전까지는 **신규 메뉴명 텍스트 기반** locator를 1순위로 사용.

```typescript
// 대메뉴 펼치기 (신규 명칭)
page.getByRole('link', { name: '라운드관리', exact: true })
page.getByRole('link', { name: '태블릿 운영 관리' })
page.getByRole('link', { name: '코스 운영 관리' })

// 하위 메뉴 (신규 명칭)
page.getByRole('link', { name: '전체 라운드' })
page.getByRole('link', { name: '캐디 리스트' })        // ← 구: 전체캐디
page.getByRole('link', { name: '메시지 기기 조회' })   // ← 구: 메시지 기록 조회
page.getByRole('link', { name: '계정 리스트' })        // ← 구: 전체 사용자 계정
```
