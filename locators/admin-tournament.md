## 화면 구성 (3영역)
1. **안내 + 웹뷰 URL 박스** (`.contents-box` 첫 박스) — 안내문구(3줄) + 리더보드/관리자 웹뷰 로그인 URL
2. **검색 영역** (`.contents-box`) — 검색어 입력 + [검색]
3. **대회 리스트 테이블** (`.contents-box`) — [신규 등록] + 15컬럼 테이블(대회별 라운드회차 다행)

> ⚠️ 안내문구가 **`.info-box-text`가 아님**(probe `info:[]`) → 표준 checkText 셀렉터 미적용. 안내 박스를 텍스트로 스코프 후 **부분 일치** 검증.

## 1. 안내 + 웹뷰 URL 박스
| 요소 | Locator | 비고 |
|------|---------|------|
| 안내문구 | 안내 박스(`.contents-box` filter `대회관리] 기능을 제공`) | 3줄. 핵심구 부분 일치: "공식적이고 다양한 방식의 대회" / "참가자, 조편성, 그룹" / "대회모드" |
| 리더보드 웹뷰 로그인 URL | label `리더보드 웹뷰 로그인 URL` + readonly input | 값 `https://smartscore.kr/leaderBoardLogin` |
| [URL복사] | 안내 박스 스코프 `getByRole('button',{name:'URL복사'})` | 클립보드 복사 — 비파괴(노출만 권장) |
| 관리자 웹뷰 로그인 URL | label `관리자 웹뷰 로그인 URL` + readonly input | 값 `https://smartscore.kr/leaderBoardLogin` |
| [보기] | 안내 박스 스코프 `getByRole('button',{name:'보기'})` | 관리자 웹뷰 진입 — 비파괴(노출만) |

> 🔴 **기획-구현 차이(추적)**: **리더보드 웹뷰 URL = 관리자 웹뷰 URL = `https://smartscore.kr/leaderBoardLogin`** 로 **동일**. 관리자 웹뷰는 별도 경로여야 자연스러움 → **복붙/URL 매핑 오류 의심**(기능 정상 여부 QA 확인 요망) → `diff` 기록.

## 3. 대회 리스트 테이블
| 요소 | Locator | 비고 |
|------|---------|------|
| [신규 등록] | `getByRole('button',{name:'신규 등록'})` | 🔴 생성 진입 — **노출만, 클릭 금지** |
| 테이블 헤더(15컬럼) | columnheader | **대회명 / 라운드회차 / 대회일자 / 주최자 / 참가자수 / 팀수 / 참가자 / 조편성 / 그룹편집 / 스코어 / 결과집계/ 출력 / 리더보드 웹뷰 / 리더보드 웹뷰 접속 인증키 / 대회 관리자 접속 인증키 / 상태** |
| 대회 행 | `tbody tr` | **≥1**(킹즈락 13행, 대회별 라운드회차 다행). 데이터 의존, 고정 count 금지 |
| 행 액션 | 행 내 버튼: `보기`(참가자/조편성/그룹편집/스코어/결과집계/리더보드) · `설정`(조편성) · `등록`(리더보드 등) · `복사`(인증키) | 🔴 전부 상세 진입/생성/복사 — **노출만, 클릭 금지** |
| 인증키 | `260424-ZOCPY` 형식 + [복사] | 데이터 의존(상태=완료 등 일부 행만 노출) |
| 상태 | `완료` 등 | 데이터 의존 |

> ⚠️ 행 액션 버튼은 대회 상태/라운드회차에 따라 노출이 다름(1R 행만 인증키·조편성 설정 노출 등) → **존재 count ≥1** 로 검증(특정 행 고정 금지).
> ℹ️ 헤더 `결과집계/ 출력` — `/` 뒤 공백 표기(`결과집계/ 출력`). columnheader 매칭은 `결과집계` 부분 일치 권장.

## 비파괴 / ⚠️ 주의
- 🔴 모든 변경/진입 동작 금지: 신규 등록·설정·등록·복사·보기(상세/웹뷰 진입) → **노출·활성만 검증**.
- 검색은 조회성(클릭 가능하나 노출 검증으로 갈음).
- 대회 행·인증키·상태는 **데이터 의존** → `≥1`/존재로 검증.
- SNB/2뎁스 토글(`snb-toggle__switchbtn`, `depth-2-toggle__switchbtn*`)은 네비 chrome — 검증 대상 아님.
- `gotoMenu` 진입 시 자동 RAW 스캔 적용.
