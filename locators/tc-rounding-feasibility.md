## ⚠️ 선행 전제 — 환경 버전 불일치

TC는 **리뉴얼(신규 UI)** 기준이나, 자동화 대상은 **td30 구버전**입니다.
→ 향후 신규 UI 배포 시 **URL 재학습 + Selector 수정 + 본 문서 갱신** 필요.

| 항목 | TC 문서(신규) | td30(구버전) | 매핑 영향 |
|------|--------------|-------------|----------|
| 검색 실행 버튼 | `[적용]` | `[검색]` (`.srch-btn`) | 명칭만 다름, 동작 동일 |
| 검색 초기화 | `[초기화]` | `[Clear]` (`.clear-btn`) | 명칭만 다름 |
| 테이블 컬럼 | 17개 | 14개 | **신규 전용 컬럼은 구버전에 없음** |
| 확인서 | 중대재해/추가/카트 확인서 분리 | `확인서` 통합 | 일부 TC 구버전 적용 불가 |
| 사진 | 사진 + 클럽사진 분리 | `사진`(통합) + 클럽체크 | 일부 TC 구버전 적용 불가 |
